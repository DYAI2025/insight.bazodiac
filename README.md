# insight.bazodiac

**Your Signal. Decoded.**

Premium astrological insight platform with luxury-tier web experience. Interactive canvas visualizations, GSAP scroll animations, and a gold-leaf neo-luxury design language.

## Development

This project uses the [AI SDLC Scaffold](https://github.com/DYAI2025/ai-scrum-scaffold) for structured, AI-first development.

### SDLC Skills

| Skill | Purpose |
|-------|---------|
| `/SDLC-init` | Project initialization |
| `/SDLC-elicit` | Requirements elicitation |
| `/SDLC-design` | Architecture & design |
| `/SDLC-decompose` | Component identification |
| `/SDLC-implementation-plan` | Task generation |
| `/SDLC-execute-next-task` | Task execution |
| `/SDLC-fix` | Bug fixes & ad-hoc changes |
| `/SDLC-status` | Project dashboard |
