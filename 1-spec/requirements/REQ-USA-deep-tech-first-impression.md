## Description

The visual design language must signal technology/fintech before astrology. Required elements: dark background (#05070a), gold accents (#D4AF37), monospace font for technical content, particle physics visualizations, glassmorphism cards. Prohibited elements: zodiac clip art, pastel gradients, stock imagery, crystal/tarot imagery, generic template layouts.

## Acceptance Criteria

- Given the hero section (before reading text), when a viewer assesses visual design, then the dominant impression is "technology/data platform" not "horoscope/wellness app"
- Given any section screenshot, when shown to a viewer without context, then the most common first descriptor is "tech" or "data" rather than "astrology" or "horoscope"
- Given the full page, when searching for prohibited visual elements, then zero instances of zodiac clip art, pastel gradients, stock photos, or template-style layouts are found
- Given comparison to Cosmic Fusion's materials, when evaluating design quality, then BAZODIAC's page is unambiguously more sophisticated
