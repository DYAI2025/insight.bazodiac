# GOAL-paid-api-readiness: Paid API Launch Readiness

**Description**: Bring FuFirE to production quality for a paid B2B API service — ensuring astronomical precision, astrological correctness, billing infrastructure, and content quality meet the standard that paying customers expect.

**Status**: Draft

**Priority**: Must-have

**Source stakeholder**: [STK-founder](../stakeholders.md)

## Success Criteria

- [ ] All P0 items (correctness, consistency, billing) implemented and tested
- [ ] All P1 items (transparency, documentation) implemented
- [ ] No known false-positive aspects due to undifferentiated orbs
- [ ] Transit calculations cover the same planet set as natal charts
- [ ] Rate-limit counters are accurate per API key
- [ ] Daily horoscope Jieqi determination uses Swiss Ephemeris, not month approximation
- [ ] All Wu-Xing parameter weights are exposed in provenance blocks
- [ ] Golden vector test suite covers all changed calculations
